# Uruchamianie backendu
* `cd backend`
* Instalacja pip'a https://www.liquidweb.com/kb/install-pip-windows/
* `pip install -r requirements.txt`
* `cd app`
* `python manage.py migrate`
* `python manage.py runserver`

# Uruchamianie frontendu
* `cd frontend`
* `npm install`
* `npm start`
